Snippet Online Shopping Website

A simple and interactive online shopping website built using HTML, CSS, and JavaScript.  
This project demonstrates core frontend development skills such as UI design, DOM manipulation, and basic e-commerce functionality.

 Features :
- Product listing and display
- Add to cart functionality
- Dynamic cart updates
- Responsive and user-friendly design
- Interactive UI with JavaScript

 Technologies Used :
- HTML – page structure
- CSS – styling and layout
- JavaScript  – interactivity and logic

 How to Run the Project :
Open home.html in your browser
